Blokschema GSM

Aan de hand van 3 gsm's kan de speler een code vinden. Deze moet vervolgens via een poster omgezet worden in morse code. Deze code wordt vervolgens doorgegeven via de ultrasone sensor. Indien de code juist is, gaat er een groen lichtje branden en opent het slot van de poliebus. Als de code fout is, gaat er een rood lichtje branden en moet de speler nog eens zijn code gaan controleren op eventuele fouten.
